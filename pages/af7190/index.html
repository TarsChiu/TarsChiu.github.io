5.清理安装包

apt-get clean
apt-get autoclean
sudo apt-get autoremove
dpkg -l |grep ^rc|awk '{print $2}' |sudo xargs dpkg -P #此命令会强力清理残留配置符号都是英文， 命令较长建议直接复制， 不要输入， 以免输错， 造成误删。其实也没必要清理它
apt-get remove [package]
apt-get remove --purge # ------(package 删除包，包括删除配置文件等) 
apt-get autoremove --purge # ----(package 删除包及其依赖的软件包+配置文件等

使用dpkg卸载

dpkg -r     #移除一个已安装的包。
dpkg -P    #完全清除一个已安装的包。和 remove 不同的是，remove 只是删掉数据和可执行文件，purge 另外还删除所有的配制文件。

# 增加grub引导界面选项

增加了重启和关机

找到/etc/grub.d

编辑40_custom

#!/bin/sh
#exec tail -n +3 $0 #attention!
# This file provides an easy way to add custom menu entries.  Simply type the
# menu entries you want to add after this comment.  Be careful not to change
# the 'exec tail' line above.

cat <<EOF
menuentry "Reboot the computer"{
	reboot
}
menuentry "Poweroff the computer"{
	halt
}
EOF

# 安装Typora

wget -qO - https://typora.io/linux/public-key.asc | sudo apt-key add -

echo -e "\ndeb https://typora.io/linux ./" | sudo tee -a /etc/apt/sources.list

sudo apt-get update

sudo apt-get install typora

进入/usr/share/applications文件夹下，找到typora.desktop文件

找到Exec=typora %U

将其修改为：

Exec=typora %U --no-sandbox

# 安装坚果云

按照官网的教程进行如下操作

wget https://www.jianguoyun.com/static/exe/installer/nutstore_linux_dist_x64.tar.gz -O /tmp/nutstore_bin.tar.gz

mkdir -p ~/.nutstore/dist && tar zxf /tmp/nutstore_bin.tar.gz -C ~/.nutstore/dist

~/.nutstore/dist/bin/install_core.sh 官网就教到这，但是你就是打不开它

但这不能阻止我们

接着找到 ~/.nutstore/dist/bin/nutstore-pydaemon.py

把python3改成python就可以启动了

ps：当你输完帐号密码登录的时候，你会发现他在功能预览那里卡住了。这是不要着急，拿着鼠标一顿狂点，或者直接把窗口关了，随你怎么乱搞，之后它自己就会好（就好还是不要结束进程）。

# 修改kali登录画面和启动画面

登录界面背景路径：
/usr/share/backgrounds/kali/

  • ps:新的kali已变为 /usr/share/backgrounds/kali/ /etc/alternatives/desktop-theme/

修改或者替换
kali-light-strips-4x3.png
kali-light-strips-16x9.png
kali-light-strips-16x10.png
这几个文件

grub2加载启动系统画面路径：
/usr/share/images/desktop-base/
创建符号链接来替换原有的desktop-grub.png的符号链接，注意！两者都是符号链接

grub2引导画面路径：
/boot/grub/themes/kali
修改或者替换
background.png
grub-4x3.png
grub-16x9.png

background.png就是grub-16x9.png可以直接复制，没问题的

最后执行update-grub即可

本人测试发现( •̀ ω •́ )✧。如果使用kali系统内自带的图片当启动画面或者锁屏壁纸的话，可以直接复制替换，不需要多加修改。

当然如果你使用自己的图片，建议与kali自带的图片参数保持一致。

# 安装GNOME桌面（不建议安装）

如果折腾后想换回原来的桌面，请按照下面的方法安装：

apt-get install gnome-core kali-defaults kali-root-login desktop-base

安装各种东西

apt install gnome-shell-extension-desktop-icons #显示桌面图标

apt install gnome-shell-extension-Clipboard Indicator #剪贴板

apt-get install gnome-tweak-tool #插件扩展

在火狐里安装相关的插件后，就可以在浏览器里直接对桌面安装插件了

插件地址 https://extensions.gnome.org

Extensions

Dash to Panel （windows风格）

Dash to Dock （mac风格）

卸载：

apt-get remove gnome-core
