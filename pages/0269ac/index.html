软件推荐集锦

# 软件推荐集锦

# Windows:

# Everything (opens new window)

快速搜索文件工具

# Ccleaner (opens new window)

清理注册表神器，可以解决网卡启动失败等等问题

# Wox (opens new window)

桌面杀手级效率神器，配合everything使用更佳

# Geek (opens new window)

软件卸载神器

# Office tool plus (opens new window)

office安装神器

# IObit Unlocker (opens new window)

解决文件被占用，无法删除神器

# Quicklook (opens new window)

效率神器 单击空格即可快速预览文件

# 天若OCR文字识别 (opens new window)

文字识别软件，超级准确的识别率

# ApowerPDF (opens new window)

PDF编辑处理神器 (opens new window)

# Mem Reduct (opens new window)

内存清理神器

# 7-zip (opens new window)

压缩软件，比bandizip还简单，无广告

# PartitionGuru (opens new window)

分区软件，比windows自带的好用多,现在改名为diskgenius (opens new window)

# Potplayer (opens new window)

超级强大的播放器，选项比较多，对小白不是很友好，但是默认设置已经够用

# Bandizip (opens new window)

无广告压缩软件

# PPT模板大全 (opens new window)

windows10应用商店下载

# Terminal (opens new window)

超好看终端 windows10应用商店下载

# 谷歌浏览器 (opens new window)

速度快，没得说

# 火狐浏览器 (opens new window)

自定义很强大，比谷歌慢一点

# Snipaste (opens new window)

截图软件

# IDM (opens new window)

下载神器，磁力链接无法下载

# AnyBizSoft – PDF to Word

PDF转Word神器

# Moo0 VideoConverter (opens new window)

视频转换器 视频格式转换

# SpeedpanX

百度网盘下载神器，付费的，价格合理

# TranslucentTB

任务栏美化

# Android:

# 倒带

简洁的开源音乐播放器

# 阅读

开源的小说阅读软件 GitHub上的MyBookshelf

# via浏览器

超级简洁的浏览器,搭配js更好用

# 酷安

一个神奇的社区

# Snapseed

照片美化软件

# WPS黑金版

无广告十分简洁

# ADM Pro

多线程下载器

# TouchRetouch

删去不需要的物体的照片处理软件

# Speedtest

测网速

# 实用工具箱

多功能的工具箱

# 存储空间清理

# 一个木函
