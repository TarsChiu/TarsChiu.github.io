Vuepress 插件安装推荐及问题指南

# Vuepress 插件安装推荐及问题指南

# 安装vuepress-plugin-element-ui

命令如下:

yarn add vuepress-plugin-element-ui -D

npm i vuepress-plugin-element-ui -D

在config.js中配置

// .vuepress/config.js
module.exports = {
  plugins: [
    'element-ui'
  ]
}

这个时候你试着运行项目,但是你发现,你的博客什么都没了!

惊不惊喜意不意外😛

别慌,element都需要一个依赖,安装好项目就可以跑起来了~

命令如下:

yarn add async-validator@1.11.5 -D

npm i async-validator@1.11.5 -D

# vuepress-plugin-one-click-copy | 代码块一键复制

yarn add vuepress-plugin-one-click-copy -D


npm i vuepress-plugin-one-click-copy -D
  • 注意这里有个奇奇怪怪的问题
["vuepress-plugin-one-click-copy",
     {
        // 代码块复制按钮
       copyMessage: "复制成功啦o(￣▽￣)ｄ",
       //showInMobile:true//只要你加上了这个，整个一键复制插件就都挂了
      },
    ],

# vuepress-plugin-mathjax | 数学公式

yarn add vuepress-plugin-mathjax -D

npm i vuepress-plugin-mathjax -D

安装上它你就可以愉快的用latex写公式啦!
